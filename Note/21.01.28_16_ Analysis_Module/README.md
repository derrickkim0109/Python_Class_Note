
# Pandas / Numpy¶
- Pandas / Numpy는 고급 데이터 북석과 수치계산등의 기능을 제공하는 확장 모듈.

- Numpy는 수치계산을 효율적으로 하기 위한 모듈로 다차원배열과 고수준의 수학 함수를 제공.
- Pandas는 테이블 분석기능을 제공하는 라이브러리
