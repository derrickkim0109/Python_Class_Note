# 자료형의 참과 거짓 
- 문자열인 경우 ""(따옴표)안에 data가 있으면 True, 없으면 False
- 리스트인 경우 []안에 data가 있으면 True, 없으면 False
- 튜플인 경우 ()안에 data가 있으면 True, 없으면 False
- 딕셔너리인 경우 {}안에 data가 있으면 True, 없으면 False
- 숫자인 경우 0이면 거짓이고, 0이 아닌 숫자는 참.
- 데이터 분석쪽에서 많이 쓰는 것이다.
